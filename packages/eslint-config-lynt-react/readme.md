# eslint-config-lynt

> The React ESLint config for [Lynt](https://github.com/saadq/lynt).

**Note**: This package is currently in beta.

## Installation

```bash
npm install eslint-config-lynt-react
```

## Usage

Put the following in your ESLint config:

```json
{
  "extends": ["lynt-react"]
}
```
